# PuzzleMind – AI Game Hub

PuzzleMind is a web-based AI game platform that includes two interactive games:
- 8 Puzzle Solver
- Tic Tac Toe AI

The project demonstrates the use of Artificial Intelligence algorithms in simple web-based games.

## Live Demo
https://amishamishra0006.github.io/PuzzleMind/

## GitHub Repository
https://github.com/amishamishra0006/PuzzleMind

## Features
- Interactive **8 Puzzle Game**
- **A* Algorithm** based puzzle solver
- **Tic Tac Toe AI opponent**
- Clean and modern UI
- Responsive web design
- Hosted online using **GitHub Pages**

## AI Concepts Used

### 1️⃣ A* Search Algorithm
Used to automatically solve the **8 Puzzle problem** by finding the optimal path from the initial state to the goal state.

### 2️⃣ Game Logic AI
Used in **Tic Tac Toe** to allow users to play against the computer.
